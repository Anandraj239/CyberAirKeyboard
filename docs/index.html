What makes it special

CORE FEATURES

🤲

Dual-Hand Tracking

Tracks both hands simultaneously via MediaPipe Hands for faster, more natural typing across the full keyboard.

🎯

Precision Pinch-to-Click

Dynamic threshold adapts to hand distance from camera — reduces misclicks whether your hand is near or far.

⚡

Debounce Cooldown

350ms per-hand cooldown with hysteresis release thresholds prevents accidental double-fires reliably.

💜

Cyber Neon UI

Semi-transparent neon aesthetic rendered natively in OpenCV — no browser, no web, fully offline and local.

📡

100% Offline

No internet required. No cloud API. Runs entirely on your local machine at real-time webcam framerate.

🚀

One-Click Launch

Double-click Run Air Keyboard.bat — it auto-creates the virtualenv, installs dependencies, and launches.

Under the hood

HOW IT WORKS

01

Webcam Capture

OpenCV reads frames from your webcam at real-time speed.

02

Hand Detection

MediaPipe Hands locates 21 landmarks on each hand per frame.

03

Hover Detection

Index fingertip (landmark 8) is matched against keyboard key rects.

04

Pinch → Keystroke

Thumb–index distance triggers pyautogui.press() system-wide.
